User Guide

This guide will walk you through how to use the Healthcare Translation Web App.

  • Voice-to-Text: Convert spoken input into text.
  • Real-Time Translation: Translate text into multiple languages.
  • Audio Playback: Listen to the translated text.
  • Translation History: View past translations.

How to Use the App

  1. Select Input Language: Choose the language of the spoken input from the Input Language dropdown.
  2. Start Speaking: Click the Start Speaking button to begin recording your voice. The app will convert your speech into text and display it in the Transcript box.
  3. Select Output Language: Choose the language you want to translate the text into from the Output Language dropdown.
  4. Translate: Click the Translate button to translate the text. The translated text will appear below the Transcript box.
  5. Listen to Translation: Click the Speak button to hear the translated text.
  6. View Translation History: Past translations are displayed in the Translation History section at the bottom of the page.

Supported Languages

  • Input Languages: English, Spanish, French, German, Chinese, e.t.c.
  • Output Languages: English, Spanish, French, German, Chinese, e.t.c.
